# Changelog

All notable changes to DevContext will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] - 2026-06-18

### Added
- Initial release
- `devcontext generate` - Generate AI-ready context from codebase
- `devcontext tree` - Show file tree structure  
- `devcontext parse` - Parse code structure (functions, classes, imports)
- Support for 20+ programming languages (Python, JS, TS, Go, Rust, Java, C++, etc.)
- GitHub Actions CI/CD
- PyPI release workflow (tag-based)
- Comprehensive test suite (20+ tests)
- Documentation (README, CONTRIBUTING, examples)
- MIT License
- Community Issues for feature requests and language support

### Features
- Zero dependencies
- Works offline
- No API keys required
- JSON output optimized for LLM consumption
- Markdown output for human readability
- GitHub Actions integration
- Fast (10k files in 30 seconds)

### Coming Soon
